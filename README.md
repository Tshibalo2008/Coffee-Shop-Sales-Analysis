# Coffee-Shop-Sales-Analysis
Bright Coffee Shop

# Introduction
This analysis is based on the Bright Coffee Shop Sales dataset, which records daily transactions from the coffee shop.
As part of supporting the new CEO’s goal to increase revenue and enhance product performance, this report provides data-driven insights and recommendations derived from historical sales data in the last six months.

# Aim
The goal of this analysis is to deliver a clear and comprehensive view of Bright Coffee Shop’s performance over the last six months.

# Focus
Product Performance

Store Performance

Marketing Strategy Analysis
